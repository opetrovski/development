Catalog Manager - PSP Integration Package

Welcome to the PSP integration package of Catalog Manager. This package helps you integrate a new payment service provider (PSP) with Catalog Manager. It provides the required Web service interfaces of Catalog Manager.

Note: The application server on which you deploy a PSP adapter must use the same JRE/JDK as the Catalog Manager server.

Package Content

This package contains the Payment service provider interface.

This interface provides the classes and methods required to implement a PSP integration adapter and the callback component for integrating a PSP with Catalog Manager. It consists of the following:

  • oscm-extsvc-psp.jar - a library containing the API for PSP integration adapters and the payment registration service
  • API javadoc - the Javadoc for the API
  • WSDL file - the WSDL file for a PSP integration adapter with its related schema file
  • Integrating a Payment Service Provider with Catalog Manager - Description of how to integrate a new payment service provider with Catalog Manager.

A sample implementation of a PSP integration adapter and callback component for integrating Heidelpay, a German PSP, can be provided on demand. Please contact your Catalog Manager support organization.
